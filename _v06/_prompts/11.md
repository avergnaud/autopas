# Prompt 11 — Anonymisation : paires original → remplacement

## Demande

Remplacer la saisie des mots-clés (textarea + checkboxes) par des lignes dynamiques à deux champs :
- champ 1 : mot-clé original (ex. "Catamania")
- champ 2 : remplacement (ex. "Fournisseur")

## Changements

### `app/services/anonymizer.py`
- `anonymize_xlsx` accepte `mapping: dict[str, str]` au lieu de `keywords: list[str]`
- Suppression de la génération automatique de tokens `[CONFIDENTIEL_N]`
- Le tri par longueur décroissante est conservé

### `app/api/web.py`
- Nouveau modèle `KeywordPair(original, replacement)`
- `AnonymizeRequest.keywords: list[KeywordPair]`
- Endpoint construit le dict `{original: replacement}` avant d'appeler `anonymize_xlsx`

### `web/private.html`
- Suppression des checkboxes métadonnées et du textarea
- Ajout de `.kw-row` (flexbox : input original → input remplacement + bouton supprimer)
- `addRow(original, replacement)` : crée une ligne dynamiquement
- `loadMetadata()` : pré-remplit les lignes avec les suggestions (original = valeur métadonnée, replacement = vide)
- `getKeywords()` retourne `[{original, replacement}]` (lignes valides seulement)
- Bouton "+ Ajouter une ligne"
