# Prompt 07

## Demande

Sur l'étape "Confirmer la structure détectée" (étape 2 du wizard), ajouter une icône "poubelle" sur chaque `<div class="structure-sheet">` (chaque onglet Excel détecté).

Comportement :
- Côté UI : supprimer la div de l'onglet dans le formulaire.
- Côté fonctionnel : l'onglet supprimé ne sera jamais traité — ni dans le parsing, ni dans la génération. Concrètement, après création de la copie de travail (working.xlsx), les onglets absents de `structure.sheets` sont physiquement supprimés du fichier.

Contrainte : ne pas modifier le fichier original (règle R-01). La suppression s'applique uniquement à `working.xlsx`.

## Implémentation

- `web/app.js` — `renderStructureForm()` : ajouter un bouton poubelle par onglet ; `deleteSheet(i)` : retire l'onglet de `state.structure.sheets` et re-rend le formulaire. Désactivé si un seul onglet reste.
- `app/services/parser_xlsx.py` — nouvelle fonction `delete_unlisted_sheets(filepath, sheet_names_to_keep)`.
- `app/services/response_generator.py` — appel à `delete_unlisted_sheets` juste après `create_working_copy`.
