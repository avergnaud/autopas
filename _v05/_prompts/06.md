## Demande

Développer l'interface web du PAS Assistant (wizard complet) + les endpoints API backend correspondants.

Avant de coder, des questions de cadrage ont été posées pour affiner les spécifications. Décisions prises :

1. **Périmètre v05** : Workflow complet — upload → confirmation structure → cadrage → anonymisation → génération → téléchargement.
2. **UX cadrage** : Wizard step-by-step (une question par écran, navigation Précédent/Suivant).
3. **Frontend** : HTML/JS/CSS vanilla (pas de framework, pas de build step).
4. **UX anonymisation** : Tableau de paires clé→valeur dynamiques (bouton Ajouter, bouton ✕ par ligne).
5. **Traitement async** : Polling toutes les 3 secondes sur `GET /api/projects/{id}/status`.
6. **Confirmation de structure** : Oui, étape simplifiée affichant les colonnes détectées.
7. **Backend en v05** : Oui, backend + frontend développés ensemble.
